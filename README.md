# AwakenCore · 给万物点睛

一个学习项目：让 AI 通过单片机控制物理世界。兼容 51 单片机、Arduino 等主流平台。

## 为什么做这个

"画龙点睛"——给龙点上眼睛，它就活了。我想做一个能让任何东西"活过来"的小硬件：插上它，灯会听你说话，电机可以远程控制，传感器能联网。

这是学习嵌入式、物联网和 AI 集成的练习项目。

## 产品定位

**OpenClaw 的物理版**——一个 Arduino 硬件兼容的智能控制器，让 AI 能直接控制物理世界。

| 对比   | OpenClaw  | AwakenCore      |
| ---- | --------- | --------------- |
| 控制对象 | 软件、文件、浏览器 | 电机、灯、传感器、继电器    |
| 形态   | 运行在电脑/服务器 | 独立硬件终端          |
| 交互   | 聊天软件、API  | 语音、App、API、聊天软件 |

## 架构设计

```
┌─────────────────────────────────────────┐
│              云端服务 (Python)            │
│   语音识别 · 大模型 · 规则引擎 · API      │
└─────────────────┬───────────────────────┘
                  │ WiFi
┌─────────────────▼───────────────────────┐
│           核心模块 (ESP32-S3)            │
│   固件 C++ · 驱动库 · Python 接口        │
│   引脚兼容 Arduino Uno                   │
└─────┬───────────┬───────────┬───────────┘
      │           │           │
   ┌──▼──┐    ┌───▼───┐   ┌───▼───┐
   │Arduino│   │51单片机│   │扩展板  │
   │(串口) │   │(串口) │   │(直插) │
   └──────┘   └───────┘   └───────┘
```

### 核心模块

- **主控**：ESP32-S3（带 WiFi）
- **引脚**：兼容 Arduino Uno，即插即用数千种扩展板
- **固件**：C++（Arduino 框架），底层驱动自研
- **接口**：暴露 Python API，降低开发门槛

### 执行器平台

| 平台                        | 连接方式             | 定位          |
| ------------------------- | ---------------- | ----------- |
| Arduino Uno / Mega / Nano | 串口 / I2C / 直插扩展板 | 开源生态，即插即用   |
| 51 单片机（STC89C52 等）        | 串口 UART          | 国内初学者友好，成本低 |
| ESP32 / ESP8266           | 原生支持             | 可独立运行       |

### 指令协议

标准化指令格式，方便各种单片机解析：

```
PIN 13 HIGH          # 数字输出
PWM 3 128            # PWM 输出（0-255）
SERVO 1 90           # 舵机角度
READ A0              # 读取模拟值
TEMP                 # 读取温度
```
